# course-typescript-contest
To install dependencies:
```bash
bun install
```
To run:
```bash
bun run index.ts
```
This project was created using `bun init` in bun v1.3.3. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

The system follows strict 1-to-many relational design, ensuring clarity and simplicity in database modeling. Authentication is handled using JWT, request validation with Zod
